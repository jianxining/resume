# Claude Code 产品页面设计分析

> 分析对象：https://claude.com/product/claude-code
> 分析日期：2026-03-02

## 一、页面构图与布局

### 1.1 整体结构
Claude Code 产品页采用**现代单页滚动布局**，主要包含以下区块：

| 区块 | 说明 |
|------|------|
| **导航栏** | 固定顶部，透明背景，滚动后变化 |
| **Hero 区** | 大标题 + 产品截图/动效 + CTA按钮 |
| **功能特性** | 网格卡片式展示核心功能 |
| **使用场景** | Tab切换或横向滚动展示不同使用方式 |
| **开发者评价** | 社交证明区域，引用真实用户反馈 |
| **CTA 底部** | 最终行动召唤 |
| **页脚** | 链接导航和品牌信息 |

### 1.2 布局特点
- **Container 系统**：使用 max-width 限制的居中容器（约 1200-1400px）
- **Grid 布局**：CSS Grid + Flexbox 混合使用
- **Breakout 布局**：支持内容突破容器全宽显示
- **响应式设计**：断点设计（small: 30em, medium: 48em, large: 62em）

### 1.3 间距系统
- 使用 CSS 变量控制全局间距
- Section 间距较大，营造呼吸感
- 组件内部使用紧凑间距

## 二、配色方案

### 2.1 主色调
Claude Code 页面使用**深色主题作为默认**，支持浅色模式切换：

#### 深色模式 (Dark Mode)
| 用途 | 色值 | 说明 |
|------|------|------|
| **背景主色** | `#0a0a0a` / `gray-950` | 接近纯黑的深灰 |
| **背景次色** | `#171717` / `gray-900` | 卡片背景 |
| **背景三级** | `#262626` / `gray-850` | 悬浮层背景 |
| **文字主色** | `#fafafa` / `gray-050` | 近乎纯白 |
| **文字次色** | `#a3a3a3` / `gray-400` | 辅助文字 |
| **文字三级** | `#737373` / `gray-500` | 禁用/提示文字 |
| **边框主色** | `#525252` / `gray-600` | 分割线 |
| **强调色** | `#C46849` | 暖橙色调，用于英雄区点缀 |

#### 浅色模式 (Light Mode)
- 以象牙白/米白为主背景
- 深灰色系作为文字颜色
- 保持相同的视觉层次逻辑

### 2.2 色彩特点
- **低饱和度**：避免鲜艳刺眼的颜色
- **高对比度**：确保文字可读性
- **橙色点缀**：品牌识别色 `#C46849` 用于关键元素
- **语义化命名**：使用 `--swatch--gray-XXX` 的 CSS 变量系统

### 2.3 按钮配色
| 类型 | 背景 | 文字 | 悬停效果 |
|------|------|------|----------|
| Primary | `gray-050` (近白) | `gray-950` (近黑) | 背景保持不变 |
| Secondary | `gray-750` | `gray-050` | 边框变化 |
| Tertiary | 透明/背景色 | `gray-050` | 边框显现 |

## 三、字体排版

### 3.1 字体选择
```css
/* 主字体 - 无衬线 */
font-family: 'Claude Sans', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;

/* 等宽字体 - 代码展示 */
font-family: 'Claude Mono', 'SF Mono', Monaco, monospace;

/* 备用字体栈 */
font-family: Arial, sans-serif; /* 基础回退 */
```

### 3.2 字体特点
- **双字体系统**：Sans-serif 用于正文，Monospace 用于代码/命令展示
- **字重范围**：400 (normal) 到 700 (bold)
- **行高优化**：使用 `lh` 单位，配合 trim 技术精确控制

### 3.3 字号层级
| 级别 | 用途 | 特点 |
|------|------|------|
| **Display/Hero** | 主标题 | 超大字号，紧凑行高，视觉冲击 |
| **H1-H6** | 章节标题 | 递减字号，清晰层级 |
| **Body Large** | 引导段落 | 较大字号，舒适阅读 |
| **Body** | 正文内容 | 标准字号，良好行高 |
| **Small/Caption** | 辅助说明 | 较小字号，次要信息 |
| **Mono** | 代码/命令 | 等宽字体，技术感 |

### 3.4 排版细节
- **Text Trim 技术**：使用 CSS 精确裁剪文字上下空白
- **Letter Spacing**：标题适度收紧，正文正常
- **Text Transform**：部分标签使用大写转换

## 四、视觉元素

### 4.1 圆角系统
- 大圆角：卡片、容器（约 12-16px）
- 中圆角：按钮、输入框（约 8px）
- 小圆角：标签、徽章（约 4px）
- 全圆角：胶囊按钮、头像（50%）

### 4.2 阴影与光效
- **Glow 效果**：品牌色发光，用于强调元素
- **柔和阴影**：浅阴影增加层次感
- **渐变背景**：微妙的渐变用于 Hero 区和特色区块

### 4.3 图标系统
- **线性图标**：简洁的 outline 风格
- **一致性**：统一 24px 或 20px 尺寸
- **交互反馈**：hover 时颜色变化

### 4.4 图片/视频处理
- **圆角容器**：媒体内容包裹在圆角容器中
- **遮罩效果**：渐变遮罩创造视觉层次
- **懒加载**：性能优化的图片加载

## 五、动效与交互

### 5.1 页面动效
- **GSAP 动画库**：复杂的滚动触发动画
- **防止闪烁**：`anti-flicker` 技术确保平滑加载
- **入场动画**：元素进入视口时的渐显/位移效果

### 5.2 微交互
- **Hover 状态**：
  - 按钮：背景色微妙变化
  - 链接：下划线动画
  - 卡片：轻微上浮 + 阴影增强
- **Focus 状态**：清晰的轮廓线，支持键盘导航
- **过渡时间**：
  - 快速反馈：100-150ms
  - 标准过渡：200-300ms
  - 复杂动画：500ms+

### 5.3 滚动体验
- **平滑滚动**：Scroll-behavior: smooth
- **视差效果**：多层背景滚动速度差异
- **固定元素**：导航栏、侧边栏智能固定

## 六、设计系统特点总结

### 6.1 核心设计理念
1. **极简主义**：去除多余装饰，聚焦内容
2. **专业可信**：深色主题传递技术专业感
3. **开发者友好**：代码展示、终端模拟等元素
4. **品牌一致性**：贯穿始终的视觉语言

### 6.2 技术实现亮点
- **CSS 变量系统**：完整的主题定制能力
- **Container Queries**：组件级响应式设计
- **Dark Mode 优先**：默认深色，支持浅色切换
- **无障碍设计**：良好的对比度和键盘导航

### 6.3 可借鉴的设计模式
| 方面 | 实践建议 |
|------|----------|
| **配色** | 使用灰色阶 + 单一强调色的方案 |
| **排版** | 建立清晰的字体层级，控制行高 |
| **布局** | 大间距营造呼吸感，网格系统保持一致 |
| **动效** | 克制使用，服务于功能而非炫技 |
| **交互** | 明确的 hover/focus 状态反馈 |
